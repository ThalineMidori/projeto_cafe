# Projeto Café

Foi realizado um estudo de caso com os dados da vendas diárias de uma loja de café. 
As informações foram obtidas pela community fabric microsoft.

Com a finalidade de colocar em prática o que estou aprendendo em Power BI, coloquei as seguintes perguntas para serem respondidas depois de uma breve análise aos dados:

- Total de vendas de cada produtos.

- Vendas por dia da semana. 

- Por ano, como foi as transações por mês?

- Qual é o lucro bruto por categoria de produto?

- Qual é a média diária de vendas.

E, com isso, foi criado um dashboard no Power Bi.
